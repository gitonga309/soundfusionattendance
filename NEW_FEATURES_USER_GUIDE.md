# Quick Start Guide - New Features

## For Salaried Employees: Self-Onboarding

### How to Complete Your Onboarding

1. **Register/Create Account**
   - Visit the registration page
   - Choose "Salaried Employee" as employment type
   - Create your account

2. **Complete Onboarding**
   - You'll be redirected to the onboarding page
   - Fill in your employment details:
     - **First Name** and **Last Name**
     - **Job Role** (Repair Technician, Driver, Accountant, Event Planner, Cleaner, Other)
     - **Monthly Salary** (in KSH)
     - **Date of Birth**
     - **National ID Number**
     - **Bank Account Number** (for salary payments)

3. **Upload Documents** (Optional but recommended)
   - **ID Photo**: Photo of your national ID
   - **Bank Details**: Bank statement or passbook copy

4. **Submit**
   - Click "Submit Application"
   - Admin will review and approve your onboarding
   - You'll get a notification when approved

5. **Check Status**
   - Go to "Onboarding Status" to check progress
   - View approval status and any rejection reasons
   - Can resubmit if rejected

## For Casual Employees: Mark Attendance with Custom Events

### How to Mark Attendance

1. **Navigate to Mark Attendance**
   - Click "Mark Attendance" from dashboard

2. **Select or Type Event**
   - Option 1: Select from dropdown list of existing events
   - Option 2: Type a custom event name if not in list
   - System will automatically create event if needed

3. **Enter Overtime Hours**
   - Enter how many hours of overtime you worked
   - Default is 0 (just base pay for the day)

4. **Submit**
   - Click "Mark Attendance"
   - Your daily pay is calculated automatically:
     - Base: KSH 1,000
     - Plus: KSH 100 per overtime hour
   - Balance updated immediately

### Editing Attendance

1. Go to "View Attendance Records"
2. Click "Edit" on today's record (only once per day)
3. Update overtime hours and/or event name
4. Can use custom event names here too

## For All Users: Mark Payments from Your Balance

### How to Mark a Payment

1. **Go to Dashboard**
   - Click "Mark Payment" button

2. **Enter Payment Details**
   - **Amount**: How much you're paying (can't exceed your balance)
   - **Payment Method**: Choose from:
     - Bank Transfer
     - Cash
     - M-Pesa
     - Other
   - **Reference Number** (Optional): 
     - For bank transfers: your transaction ID or account reference
     - For M-Pesa: M-Pesa reference number
     - For checks: check number
   - **Additional Notes** (Optional): Any other relevant info

3. **Confirm**
   - Click "Confirm Payment"
   - Your balance is immediately reduced
   - Payment is recorded in the system

4. **View Payment History**
   - All your payments are listed on dashboard
   - Shows date, amount, method, and reference number

## For All Users: Reimbursement Without Event or Receipt

### How to Request Reimbursement

1. **Go to "Request Reimbursement"**
   - From dashboard menu

2. **Fill in Details**
   - **Expense Type**: 
     - Transport (Uber/Bolt)
     - Purchase (Equipment/Supplies)
     - Airtime
     - Meal/Food
     - Other
   - **Amount**: How much you're claiming (KSH)
   - **Description**: Explain the expense
   - **Event** (Optional): 
     - Can leave blank if expense wasn't for a specific event
   - **Receipt/Image** (Optional):
     - Can submit without receipt if needed
     - But receipt is recommended for faster approval

3. **Submit**
   - Click "Submit Request"
   - Admin will review
   - You'll see approval/rejection status in "My Reimbursements"

## Tips & Best Practices

### For Employees

1. **Attendance**
   - Mark attendance same day you work
   - Use custom event name if your event not in list
   - You can edit once per day

2. **Reimbursement**
   - Keep receipts for important expenses
   - Add detailed description
   - Attach receipt if possible (faster approval)
   - Can still claim without receipt if needed

3. **Payments**
   - Mark payment same day you transfer funds
   - Add reference number for tracking
   - Keep payment confirmations

4. **Balance Checking**
   - Check dashboard regularly
   - All balance changes are logged
   - Can verify payment history anytime

### For Admin

1. **Reimbursement Approval**
   - Review monthly
   - Approve immediately for faster payment
   - Use bulk actions to approve multiple at once
   - Always check for receipt if available

2. **User History**
   - View history for disputes or audits
   - See complete audit trail
   - Verify all changes are accurate

3. **Balance Management**
   - Monitor balance changes
   - Adjust if there are errors
   - Document reasons for adjustments

## Troubleshooting

### Event Not in Dropdown?
- Type the event name in the text field
- System will create it automatically
- You can use custom event names

### Can't Submit Reimbursement?
- Check amount is > 0 and â‰¤ 50,000
- Expense type must be selected
- Description can be optional
- Try again (event and receipt are optional)

### Balance Not Updating?
- Payment takes effect immediately after marking
- Reimbursement takes effect when approved by admin
- Check "Balance Changes" section for details
- Contact admin if incorrect

### Can't Edit Attendance?
- Can only edit once per day
- Edit must be same day as attendance
- Contact admin for corrections after one edit
